# SETUP AFTER CLONING
## Open terminal and run these commands.
1. Run "npm install" and "composer install".
2. Run "php artisan key:generate".
3. Run "php artisan migrate" to setup DB.
4. Run "composer run dev".

## Notes:
- We are using Laravel 12.
- We are using Flowbite v4 for templates and TailwindCSS v4.

## Optional:
### Follow these steps if you have a problem.
- Use Laragon 6.0.
- Use the MySQL database from Laragon.
- Use PHP 8.4.16.
